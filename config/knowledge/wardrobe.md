# 服饰与换装业务知识

- 用户提出换装要求不代表角色已经同意，也不代表换装已经发生。
- 回复明确表示已经穿上、脱下或换好时，必须提交对应的已完成 `state_ops`。
- 当前服饰按 `upper`、`lower`、`legwear`、`footwear`、`accessories` 管理；每个槽位从内到外分层。
- 脱掉一件衣物只移除对应槽位的目标层，不要重新描述或重建整套服饰。
- 移除外层不会自动移除内层；已经移除的衣物不会自动重新穿回。
- 连体衣物可以同时占据 `upper` 和 `lower`，移除时作为同一件衣物原子移除。
- 完整换装才使用 `replace`，并给出完整 garment 集合；单件穿脱使用 `wear` 或 `remove`。
- 服饰风格、颜色、材质和装饰使用英文 SD 标签，但状态层级不依赖具体风格名称。
- 搭配应符合地点、时间、活动和刚才的对话，不要为了性感随机增加丝袜、高跟鞋或暴露单品。
- 角色可以评价用户提出的搭配，认为不协调时可以拒绝或提出更合适的方案。
- `footwear` 和 `legwear` 都为空才表示裸足；不要把脱鞋等同于脱袜。
- 图片只能读取提交后的服饰快照，不能临时增加、删除或恢复衣物。
